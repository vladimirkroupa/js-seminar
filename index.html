Invocation

Apart from arguments parameter, function receives this parameter. Value of this is determined by invocation pattern.

1. Method Invocation Pattern

var a = { fun : console.log };

a.fun('a');

Function stored as property of an object is a method.

When a method is invoked, this is bound to that object.

  • method invocation

2. Function Invocation Pattern

When function is not a property of an object, it is invoked as a function.

var sum = add(3, 4);

When function is invoked with this pattern, this is bound to the global object. A consequence of this is that a method cannot employ an inner function to help it.

Closure

The closure of the generated functions contain pointers to variables, not a snapshot of the values at the point of definition, meaning that if variables change, these changes are visible from within the function.
